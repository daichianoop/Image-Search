About
This is a simple Image Search Application made by User1 using Unsplash API, Tailwind CSS and Vanilla Javascript.
This project uses concept of various Methods such as appendChild(); Method, tryCatch(); Method, etc. using map(); and async functions.
For more information on Unsplash API , visit → Unsplash.com .
